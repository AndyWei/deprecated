### This file contains all curl commands used for manual test

## signin (GET a token)
curl -i -u jack@gmail.com:password "http://localhost:8000/v1/signin"


## person
# GET a person's own profile
curl -i -H "Authorization: Bearer 5dHwvDMJacb2umovRvIn" "http://localhost:8000/v1/person/profile"

# GET a another person's profile
curl -i -H "Authorization: Bearer 5dHwvDMJacb2umovRvIn" "http://localhost:8000/v1/person?id=1"

# Update a person's profile
curl -i -H "Authorization: Bearer 5dHwvDMJacb2umovRvIn" --data "yob=1995&name='dodo'&gender=2&bio='sound of silence'" "http://localhost:8000/v1/person/profile"



## comments
# POST a comment
curl -i -X POST -H "Authorization: Bearer GiDsFzn99eaYKIFLk0QZ" --data "order_id=1&body=@andy @ping yes" "http://localhost:8000/v1/comments"
